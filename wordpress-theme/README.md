## Description

Arms Complex is a premium WordPress theme designed specifically for firearms and ammunition e-commerce stores. Built on WooCommerce with advanced product variations, ballistic calculators, and comprehensive filtering.

## Features

✅ Full WooCommerce integration  
✅ Quantity-based product variations  
✅ Advanced product filtering  
✅ Wishlist functionality  
✅ Product comparison  
✅ Ballistic calculator  
✅ Reloading guide  
✅ Load recipes system  
✅ Responsive design  
✅ Dark mode support  
✅ Multiple layout variations  
✅ Demo content import  
✅ Customizer options  

## Installation

1. Download the theme zip file
2. Go to WordPress Admin → Appearance → Themes → Add New → Upload Theme
3. Upload the zip file and activate
4. Install and activate WooCommerce plugin
5. Import demo content when prompted

## Theme Setup

### Required Plugins
- **WooCommerce** (required)
- WordPress Importer (recommended for demo import)

### Initial Configuration

1. **Import Demo Content**: Click the admin notice to import sample products and pages
2. **Configure WooCommerce**: Go to WooCommerce → Settings and configure your store
3. **Customize Theme**: Go to Appearance → Customize to set logo, colors, and options
4. **Set Menus**: Go to Appearance → Menus to create navigation menus
